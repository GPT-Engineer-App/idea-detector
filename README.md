# idea-detector

Build me an AI-powered business problem identification Micro SaaS.  The AI should comb through social media, online forums, review sites, blogs, etc, to identify salient pain-points or Jobs to be done along with existing solutions, and produce back sets of viable problems to go after and solve for entrepreneurs.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/idea-detector.git
cd idea-detector
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
